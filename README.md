## ğŸ“ Customization

### Update Personal Information
- Edit `index.html` to update your name, bio, and contact details
- Replace images in the `image/` folder
- Update your CV in `assets/CV.pdf`

### Modify Colors
Edit CSS variables in `style.css`:
```css
:root {
  --bg-color: #1f242d;
  --second-bg-color: #323946;
  --text-color: #fff;
  --main-color: #0ef;
  --accent-color: #00d9ff;
}
```

### Contact Form Setup
Replace the Google Sheets script URL in `index.html`:
```javascript
const scriptURL = 'YOUR_GOOGLE_SCRIPT_URL_HERE';
```
